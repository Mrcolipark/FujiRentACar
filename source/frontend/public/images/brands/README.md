# 品牌 Logo 图片说明

请在此目录放置以下 8 个品牌的 Logo 图片：

## 需要的文件：

1. **mercedes.png** - 奔驰 (Mercedes-Benz)
2. **maserati.png** - 玛莎拉蒂 (Maserati)
3. **toyota.png** - 丰田 (Toyota)
4. **porsche.png** - 保时捷 (Porsche)
5. **lexus.png** - 雷克萨斯 (Lexus)
6. **bmw.png** - 宝马 (BMW)
7. **mazda.png** - 马自达 (Mazda)
8. **honda.png** - 本田 (Honda)

## 图片要求：

- **格式**: PNG（推荐透明背景）
- **尺寸**: 建议 500x500 到 1000x1000 像素
- **背景**: 透明背景最佳
- **质量**: 高清矢量或高分辨率

## Logo 获取建议：

1. 访问各品牌官网的媒体资料页面下载官方 Logo
2. 使用品牌资源网站（如 brandfetch.com）
3. 搜索 "[品牌名] logo png transparent"

```

├── toyota.png

```
